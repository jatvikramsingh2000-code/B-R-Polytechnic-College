# B-R-Polytechnic-College

A simple, modern college website for students, management, notices, and contact communication.

## Pages
- Home landing page
- Student portal page
- Management dashboard page
- Notices page
- Contact page

## Run locally
Open the project folder in a browser or run:

```bash
python3 -m http.server 3000
```

Then visit http://127.0.0.1:3000/
